**Risk acknowledgment (addresses review concern #4):** Programmatic generation is used because Claude cannot run the TouchOSC GUI editor. The .tosc format is zlib-compressed XML with an undocumented schema. Mitigations: (1) the script validates output by decompressing and checking key addresses, (2) the human verification checkpoint in Task 2 tests actual TouchOSC import, (3) the script is kept as a reproducible build tool so the user can regenerate after manual edits in the editor.

**CRITICAL first step:** Before writing any generation code, read the actual TouchOSC .tosc XML format. Study these resources:
1. Read `docs/osc.md` for the authoritative address list
2. Search online or use existing knowledge of the TouchOSC XML node schema
3. If available, examine any existing .tosc files for reference

The executor MUST study the actual .tosc XML format carefully. The tosclib Python library documents the format as:
- Root: `<lexml version="N">`
- Nodes: `<node type="TYPE">` where TYPE is PAGE, GROUP, FADERV, FADERH, BUTTON, LABEL, ENCODER, TEXT, etc.
- Properties: `<property type="TYPE"><key>KEY</key><value>VALUE</value></property>` where type is s (string), i (int), f (float), b (bool), r (rect), c (color)
- Frame (position/size): `<property type="r"><key>frame</key><value><x>X</x><y>Y</y><w>W</w><h>H</h></value></property>`
- Color: `<property type="c"><key>color</key><value><r>R</r><g>G</g><b>B</b><a>A</a></value></property>`
- OSC messages: `<messages><osc><enabled>1</enabled><send>1</send><receive>1</receive><feedback>0</feedback><connections>11111</connections><triggers>/any/x</triggers><path>/JamWide/...</path></osc></messages>`

**If the executor is uncertain about the exact XML schema**, use `pip install tosclib` and use the tosclib API to generate valid nodes. This is safer than guessing the XML format:

**Script structure:**

```python
#!/usr/bin/env python3
"""Generate JamWide.tosc TouchOSC template.

Run: python3 scripts/generate_tosc.py
Output: assets/JamWide.tosc

This is a build-time utility for reproducible template generation.
The primary asset is assets/JamWide.tosc. Users can also edit the
template in TouchOSC Editor and re-import; this script can regenerate
the baseline if needed.
"""
import xml.etree.ElementTree as ET
import zlib
import os
import sys

# ... helper functions for creating TouchOSC XML nodes ...
# ... build_template() function that creates all controls ...
# ... main() that writes compressed XML to assets/JamWide.tosc ...

def main():
    root = build_template()
    xml_bytes = ET.tostring(root, encoding="utf-8", xml_declaration=True)
    compressed = zlib.compress(xml_bytes)

    script_dir = os.path.dirname(os.path.abspath(__file__))
    project_root = os.path.dirname(script_dir)
    output_dir = os.path.join(project_root, "assets")
    os.makedirs(output_dir, exist_ok=True)
    output_path = os.path.join(output_dir, "JamWide.tosc")

    with open(output_path, "wb") as f:
        f.write(compressed)

    # Self-validation: decompress and check key addresses
    xml_str = zlib.decompress(compressed).decode("utf-8")
    required_addresses = [
        "/JamWide/master/volume",
        "/JamWide/master/mute",
        "/JamWide/metro/volume",
        "/JamWide/local/1/volume",
        "/JamWide/local/4/volume",
        "/JamWide/remote/1/volume",
        "/JamWide/remote/8/volume",
        "/JamWide/remote/1/name",
        "/JamWide/session/bpm",
        "/JamWide/session/connect",
        "/JamWide/session/disconnect",
        "/JamWide/master/vu/left",
    ]
    missing = [a for a in required_addresses if a not in xml_str]
    if missing:
        print(f"ERROR: Missing addresses in template: {missing}", file=sys.stderr)
        sys.exit(1)

    # Verify remote VU is NOT in template (intentional omission)
    if "/JamWide/remote/1/vu/" in xml_str:
        print("WARNING: Remote VU found in template (should be omitted per design)", file=sys.stderr)

    # Verify no out-of-range remote slots
    if "/JamWide/remote/9/volume" in xml_str:
        print("ERROR: Template has remote slot 9+ (should have max 8 per D-13)", file=sys.stderr)
        sys.exit(1)

    print(f"Generated {output_path} ({len(compressed)} bytes)")
    print(f"Validated {len(required_addresses)} required addresses")

if __name__ == "__main__":
    main()
```
