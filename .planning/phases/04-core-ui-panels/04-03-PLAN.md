**1. Create `juce/ui/VuMeter.h` and `juce/ui/VuMeter.cpp`:**

VuMeter is a custom-painted component with 30Hz timer, segmented LED rendering per UI-SPEC.

```cpp
// VuMeter.h
#pragma once
#include <JuceHeader.h>

class VuMeter : public juce::Component, private juce::Timer
{
public:
    VuMeter();
    ~VuMeter() override;

    void setLevels(float left, float right);
    void paint(juce::Graphics& g) override;

private:
    void timerCallback() override;
    void paintBar(juce::Graphics& g, juce::Rectangle<float> bounds, float level);

    float targetLeft = 0.0f, targetRight = 0.0f;
    float displayLeft = 0.0f, displayRight = 0.0f;

    static constexpr float kAttack = 0.8f;
    static constexpr float kRelease = 0.92f;
    static constexpr int kSegmentHeight = 7;
    static constexpr int kSegmentGap = 2;

    JUCE_DECLARE_NON_COPYABLE_WITH_LEAK_DETECTOR(VuMeter)
};
```

VuMeter.cpp implementation:
- Constructor: `startTimerHz(30)`
- Destructor: `stopTimer()`
- `setLevels(float l, float r)`: Store in targetLeft/targetRight.
- `timerCallback()`: Apply ballistics per UI-SPEC:
  - displayLeft = (targetLeft > displayLeft) ? targetLeft * kAttack + displayLeft * (1-kAttack) : displayLeft * kRelease
  - Same for right channel
  - Call `repaint()`
- `paint()`: 
  - Total width = getWidth(). Each channel bar = (width - 6) / 2 pixels wide. 6px gap between.
  - Fill background with `kVuBackground` (#0A0D18)
  - Left bar: x=0, width=(w-6)/2. Right bar: x=(w-6)/2+6, width=(w-6)/2.
  - Call `paintBar()` for each.
- `paintBar()`:
  - Calculate total segments = bounds.getHeight() / (kSegmentHeight + kSegmentGap)
  - For each segment from bottom to top (index 0 = bottom):
    - segmentY = bounds.getBottom() - (i+1) * (kSegmentHeight + kSegmentGap)
    - normalizedPosition = (float)(i+1) / totalSegments  (0.0 at bottom, 1.0 at top)
    - isLit = normalizedPosition <= level
    - If lit: color based on normalizedPosition:
      - 0.0-0.7: `kVuNominal` (#33CC33)
      - 0.7-0.9: `kVuWarm` (#E6B333)
      - 0.9-1.0: `kVuClip` (#E63333)
    - If unlit: `kVuUnlit` (#0F1520)
    - Draw filled rounded rect (radius 1px) for each segment

BeatBar is a 22px-tall custom-painted component showing beat/interval progress per UI-SPEC.

```cpp
// BeatBar.h
#pragma once
#include <JuceHeader.h>

class BeatBar : public juce::Component
{
public:
    BeatBar();

    void update(int bpi, int currentBeat, int intervalPos, int intervalLen);
    void paint(juce::Graphics& g) override;

private:
    int bpi_ = 0;
    int currentBeat_ = 0;
    int intervalPos_ = 0;
    int intervalLen_ = 0;

    JUCE_DECLARE_NON_COPYABLE_WITH_LEAK_DETECTOR(BeatBar)
};
```

BeatBar.cpp:
- `update()`: Store values, repaint.
- `paint()`:
  - If bpi_ <= 0 or intervalLen_ <= 0: fill with `kVuBackground` (#0A0D18), return.
  - Calculate segment width = (float)getWidth() / bpi_
  - For each beat i (0 to bpi_-1):
    - segX = i * segWidth, segW = segWidth - 1 (1px gap)
    - If i < currentBeat_ (past): fill with `kAccentConnect` (#40E070) at 35% opacity -> `Colour(0xff40E070).withAlpha(0.35f)`
    - If i == currentBeat_ (current): fill with `kAccentConnect` (#40E070) at 100% opacity. Draw beat number centered in segment, 11px bold white font.
    - If i > currentBeat_ (future): fill with `kVuBackground` (#0A0D18)
    - Downbeat accent: if (i % 4 == 0), draw 2px tall bar at top of segment in `kAccentConnect` at 60% opacity
    - For downbeats (i % 4 == 0) that are not the current beat: draw beat number in `kTextSecondary` at 9px font
  - BPI adaptation per UI-SPEC:
    - BPI <= 8: all segments numbered
    - BPI 9-24: standard (current + downbeats numbered)
    - BPI >= 32: only downbeat group markers, segments thinner

ChannelStripArea is a container that manages local + remote ChannelStrips and a master strip, with horizontal scrolling for overflow.

```cpp
// ChannelStripArea.h
#pragma once
#include <JuceHeader.h>
#include "ChannelStrip.h"
#include "ui/ui_state.h"
#include <vector>

class JamWideJuceProcessor;

class ChannelStripArea : public juce::Component
{
public:
    explicit ChannelStripArea(JamWideJuceProcessor& processor);

    void refreshFromUsers(const std::vector<UiRemoteUser>& users);
    void updateVuLevels();  // Called from timer to update VU meters
    void setDisconnectedState();
    void setConnectedState();

    void paint(juce::Graphics& g) override;
    void resized() override;

    // Callbacks for editor
    std::function<void(const juce::String&)> onServerAddressFilled;  // not used yet

    int getDesiredWidth() const;  // For dynamic window width

private:
    void rebuildStrips(const std::vector<UiRemoteUser>& users);

    JamWideJuceProcessor& processorRef;

    ChannelStrip localStrip;
    ChannelStrip masterStrip;
    std::vector<std::unique_ptr<ChannelStrip>> remoteStrips;

    juce::Viewport viewport;
    juce::Component stripContainer;  // Lives inside viewport

    juce::Label emptyStateLabel;  // "Connect to a server to start jamming"
    bool isConnected = false;

    static constexpr int kStripWidth = 100;
    static constexpr int kStripPitch = 106;  // 100px + 6px gap
    static constexpr int kMasterWidth = 110;
    static constexpr int kMaxVisibleStrips = 13;

    JUCE_DECLARE_NON_COPYABLE_WITH_LEAK_DETECTOR(ChannelStripArea)
};
```

ChannelStripArea.cpp:

Constructor:
- processorRef stored
- Configure localStrip: `localStrip.configure(ChannelStrip::StripType::Local, "Local", "FLAC", 1, false)`
- Configure masterStrip: `masterStrip.configure(ChannelStrip::StripType::Master, "MASTER")`
- Set up viewport: `viewport.setViewedComponent(&stripContainer, false)`. Vertical scroll off, horizontal scroll auto. Scrollbar at bottom, 14px height. Dark theme colors.
- emptyStateLabel: "Connect to a server to start jamming" per D-28. 16px font, `kTextSecondary`, centered. addAndMakeVisible.
- Set disconnected state initially.

`refreshFromUsers(users)`:
- Clear existing remoteStrips
- For each user in users:
  - If user has 1 channel: create single ChannelStrip with `StripType::Remote`
  - If user has N>1 channels: create parent strip with expand button + N child strips with `StripType::RemoteChild`
  - Configure each with name, codec string (derive from codec_fourcc: 'FLAC'=>"FLAC", 'OGGv'=>"Vorbis", else "?"), channelCount, expanded state
  - Wire onSubscribeToggled to push `SetUserChannelStateCommand` to cmd_queue
  - Wire onRoutingChanged similarly (Phase 6 wires actual routing)
- Call rebuildStrips() to layout

`rebuildStrips(users)`:
- Clear stripContainer children
- Add localStrip to stripContainer
- Add each remote strip to stripContainer
- Add masterStrip to stripContainer
- Call resized() to layout

`updateVuLevels()`:
- localStrip.setVuLevels(processorRef.uiSnapshot.local_vu_left.load(relaxed), processorRef.uiSnapshot.local_vu_right.load(relaxed))
- masterStrip.setVuLevels(processorRef.uiSnapshot.master_vu_left.load(relaxed), processorRef.uiSnapshot.master_vu_right.load(relaxed))
- Remote VU: For now, remote VU levels are 0 (reading remote VU requires NJClient API calls that must happen in the run thread; this will be wired when mixer controls are added in Phase 5).

`setDisconnectedState()`:
- isConnected = false
- Hide localStrip, masterStrip, viewport
- Show emptyStateLabel with "Connect to a server to start jamming"
- Clear remote strips
- repaint()

`setConnectedState()`:
- isConnected = true
- Show localStrip, masterStrip, viewport
- Hide emptyStateLabel
- resized()

`paint()`:
- Fill background with `kBgPrimary` (#1A1D2E)
- If !isConnected: draw empty state (handled by emptyStateLabel)

`resized()`:
- If !isConnected: center emptyStateLabel in bounds. Return.
- Layout inside stripContainer (horizontal):
  - localStrip at x=0, width=kStripWidth
  - Each remote strip at x += kStripPitch
  - Total content width = (1 + remoteStrips.size()) * kStripPitch
  - stripContainer.setSize(totalContentWidth, getHeight())
- Master strip pinned to right edge of ChannelStripArea (not inside viewport, always visible per UI-SPEC)
- viewport: x=0, width = getWidth() - kMasterWidth, full height. Shows stripContainer.
- masterStrip: x = getWidth() - kMasterWidth, width = kMasterWidth, full height.
- All strips: full height of this component.

`getDesiredWidth()`: Return (1 + remoteStrips.size()) * kStripPitch + kMasterWidth. Clamped to kMaxVisibleStrips * kStripPitch + kMasterWidth max.
