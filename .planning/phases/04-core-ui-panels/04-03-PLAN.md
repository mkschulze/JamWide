REVIEW FIX #7: VuMeter has NO internal timer. It exposes `setLevels()` and `tick()` methods. The parent ChannelStripArea runs a single centralized 30Hz timer that calls tick() on all meters.

```cpp
// VuMeter.h
#pragma once
#include <JuceHeader.h>

class VuMeter : public juce::Component
{
public:
    VuMeter();

    // Called by external code to set target levels
    void setLevels(float left, float right);

    // Called by centralized timer to apply ballistics and repaint
    // REVIEW FIX #7: No internal timer -- parent drives updates
    void tick();

    void paint(juce::Graphics& g) override;

private:
    void paintBar(juce::Graphics& g, juce::Rectangle<float> bounds, float level);

    float targetLeft = 0.0f, targetRight = 0.0f;
    float displayLeft = 0.0f, displayRight = 0.0f;

    static constexpr float kAttack = 0.8f;
    static constexpr float kRelease = 0.92f;
    static constexpr int kSegmentHeight = 7;
    static constexpr int kSegmentGap = 2;

    JUCE_DECLARE_NON_COPYABLE_WITH_LEAK_DETECTOR(VuMeter)
};
```

VuMeter.cpp implementation:
- Constructor: no timer start (REVIEW FIX #7)
- `setLevels(float l, float r)`: Store in targetLeft/targetRight.
- `tick()`: Apply ballistics per UI-SPEC:
  - displayLeft = (targetLeft > displayLeft) ? targetLeft * kAttack + displayLeft * (1-kAttack) : displayLeft * kRelease
  - Same for right channel
  - Call repaint()
- `paint()`:
  - Total width = getWidth(). Each channel bar = (width - 6) / 2 pixels wide. 6px gap between.
  - Fill background with kVuBackground (#0A0D18)
  - Left bar: x=0, width=(w-6)/2. Right bar: x=(w-6)/2+6, width=(w-6)/2.
  - Call paintBar() for each.
- `paintBar()`:
  - totalSegments = bounds.getHeight() / (kSegmentHeight + kSegmentGap)
  - For each segment from bottom to top (index 0 = bottom):
    - segmentY = bounds.getBottom() - (i+1) * (kSegmentHeight + kSegmentGap)
    - normalizedPosition = (float)(i+1) / totalSegments
    - isLit = normalizedPosition <= level
    - If lit: 0.0-0.7 -> kVuNominal (#33CC33), 0.7-0.9 -> kVuWarm (#E6B333), 0.9-1.0 -> kVuClip (#E63333)
    - If unlit: kVuUnlit (#0F1520)
    - Draw filled rounded rect (radius 1px)

`timerCallback()` -- REVIEW FIX #7: single centralized timer:
```cpp
void ChannelStripArea::timerCallback()
{
    if (!isConnected) return;

    // Update VU target levels from processor data
    updateVuLevels();

    // Tick all VU meters (apply ballistics + repaint)
    localStrip.tickVu();
    masterStrip.tickVu();
    for (auto& strip : remoteStrips)
        strip->tickVu();
}
```

`updateVuLevels()`:
```cpp
// Local VU from atomic snapshot
localStrip.setVuLevels(
    processorRef.uiSnapshot.local_vu_left.load(std::memory_order_relaxed),
    processorRef.uiSnapshot.local_vu_right.load(std::memory_order_relaxed));

// Master VU from atomic snapshot
masterStrip.setVuLevels(
    processorRef.uiSnapshot.master_vu_left.load(std::memory_order_relaxed),
    processorRef.uiSnapshot.master_vu_right.load(std::memory_order_relaxed));

// REVIEW FIX #6: Remote VU from cachedUsers -- NOT hardcoded zero.
// cachedUsers is populated by run thread via GetRemoteUsersSnapshot()
// which includes vu_left/vu_right per channel.
const auto& users = processorRef.cachedUsers;
int stripIdx = 0;
for (const auto& user : users) {
    for (const auto& ch : user.channels) {
        if (stripIdx < static_cast<int>(remoteStrips.size())) {
            remoteStrips[stripIdx]->setVuLevels(ch.vu_left, ch.vu_right);
        }
        ++stripIdx;
    }
}
```

`refreshFromUsers(users)`:
- Clear existing remoteStrips
- For each user in users:
  - If 1 channel: create single ChannelStrip Remote
  - If N>1: create parent + N child RemoteChild strips
  - Derive codec string from codec_fourcc if available, else show nothing
  - Wire onSubscribeToggled to push SetUserChannelStateCommand
- Call rebuildStrips()

`rebuildStrips()`:
- Clear stripContainer, add localStrip + remote strips (NOT masterStrip -- master is outside viewport)
- Call resized()

`setDisconnectedState()` / `setConnectedState()`:
- Toggle isConnected, show/hide strips vs empty state

`resized()`:
- If !isConnected: center emptyStateLabel + browseButton below
- If connected: layout strips in stripContainer, master pinned right outside viewport

`getDesiredWidth()`: (1 + remoteStrips.size()) * kStripPitch + kMasterWidth, clamped
