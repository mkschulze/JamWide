  <action>
  1. Add git submodules:
     - `git submodule add https://github.com/juce-framework/JUCE.git libs/juce` then checkout tag `8.0.12`
     - `git submodule add https://github.com/free-audio/clap-juce-extensions.git libs/clap-juce-extensions` then `cd libs/clap-juce-extensions && git submodule update --init --recursive`

  2. Add `set(CMAKE_EXPORT_COMPILE_COMMANDS ON)` near the top (after `CMAKE_POSITION_INDEPENDENT_CODE`). Generates compile_commands.json for clangd/IDE autocompletion.

  3. Modify CMakeLists.txt to add two build options:
     - `option(JAMWIDE_BUILD_JUCE "Build JUCE plugin targets" ON)`
     - `option(JAMWIDE_BUILD_CLAP "Build CLAP/ImGui plugin targets" ON)` -- wrap ALL existing CLAP-related code (imgui, jamwide-ui, jamwide-impl, make_clapfirst_plugins, etc.) inside `if(JAMWIDE_BUILD_CLAP)` ... `endif()`. This ensures the CLAP build is still the default but can be disabled.

  4. Define plugin identity as CMake variables (single source of truth, referenced by both juce_add_plugin and clap_juce_extensions_plugin — pattern from PhaseGrid reference):
     ```cmake
     set(JAMWIDE_JUCE_BUNDLE_ID "com.jamwide.juce-client")
     set(JAMWIDE_JUCE_MANUFACTURER_CODE "JmWd")
     set(JAMWIDE_JUCE_PLUGIN_CODE "JwJc")
     ```

  5. Inside `if(JAMWIDE_BUILD_JUCE)` block, add:
     - `add_subdirectory(libs/juce EXCLUDE_FROM_ALL)`
     - `juce_add_plugin(JamWideJuce ...)` with these settings:
       - PRODUCT_NAME "JamWide JUCE"
       - COMPANY_NAME "JamWide"
       - BUNDLE_ID "${JAMWIDE_JUCE_BUNDLE_ID}"
       - PLUGIN_MANUFACTURER_CODE ${JAMWIDE_JUCE_MANUFACTURER_CODE}
       - PLUGIN_CODE ${JAMWIDE_JUCE_PLUGIN_CODE}

       - VST3_CATEGORIES Fx Network

       - NEEDS_MIDI_OUTPUT FALSE
       - IS_SYNTH FALSE
       - EDITOR_WANTS_KEYBOARD_FOCUS TRUE
       - VST3_AUTO_MANIFEST FALSE
       - COPY_PLUGIN_AFTER_BUILD TRUE
       - MICROPHONE_PERMISSION_ENABLED TRUE
       - MICROPHONE_PERMISSION_TEXT "JamWide needs microphone access for standalone mode"
     - `juce_generate_juce_header(JamWideJuce)`
     - target_sources: juce/JamWideJuceProcessor.cpp, juce/JamWideJuceEditor.cpp
     - target_compile_definitions: JUCE_WEB_BROWSER=0, JUCE_USE_CURL=0, JUCE_VST3_CAN_REPLACE_VST2=0, JUCE_DISPLAY_SPLASH_SCREEN=0

     ```cmake
             foreach(_fmt Standalone VST3 AU CLAP)

                     add_custom_command(TARGET JamWideJuce_${_fmt} POST_BUILD
                         COMMAND /usr/bin/codesign --force --sign "-"
                                 --timestamp=none --deep "$<TARGET_BUNDLE_DIR:JamWideJuce_${_fmt}>"
                         COMMENT "Dev codesign JamWideJuce_${_fmt}"
                         VERBATIM
                     )
                 endif()
             endforeach()
         endif()
     endif()
     ```

  **JamWideJuceProcessor.cpp:**
  - Constructor initializes APVTS with createParameterLayout()
  - `processBlock()`: Clear all output channels (produce silence). Use `juce::ScopedNoDenormals`. Add comment: "Phase 3 will integrate NJClient::AudioProc here"
  - `isBusesLayoutSupported()`: Main output must be stereo, main input must be stereo, all other buses must be stereo or disabled
  - `getStateInformation()` / `setStateInformation()`: Serialize/deserialize APVTS state via XML. Include a `stateVersion` integer property (initial value 1) in the XML root — this enables backward-compatible state migration in future phases (pattern from PhaseGrid reference plugin which is now on version 3). Example:
    ```cpp
    void getStateInformation(juce::MemoryBlock& destData)
    {
        auto state = apvts.copyState();
        state.setProperty("stateVersion", 1, nullptr);
        std::unique_ptr<juce::XmlElement> xml(state.createXml());
        copyXmlToBinary(*xml, destData);
    }
    void setStateInformation(const void* data, int sizeInBytes)
    {
        std::unique_ptr<juce::XmlElement> xml(getXmlFromBinary(data, sizeInBytes));
        if (xml && xml->hasTagName(apvts.state.getType()))
        {
            auto tree = juce::ValueTree::fromXml(*xml);
            int version = tree.getProperty("stateVersion", 0);
            // Future phases: migrate state based on version
            apvts.replaceState(tree);
        }
    }
    ```
  - Other overrides: minimal valid implementations (getName returns "JamWide JUCE", acceptsMidi/producesMidi/isMidiEffect return false, getTailLengthSeconds returns 0, etc.)
  - Add a static constexpr `int currentStateVersion = 1;` to the class header for reference
