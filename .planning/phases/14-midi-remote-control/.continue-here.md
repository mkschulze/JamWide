## Critical Anti-Patterns

| Pattern | Severity | Description |
|---------|----------|-------------|
| Audio-thread callback to message-thread data | blocking | MidiLearnManager::tryLearn callback must NEVER modify staging_ or call repaint from audio thread. Always use callAsync. |
| MIDI gated behind connection | blocking | processIncomingMidi/appendFeedbackMidi must run unconditionally in processBlock, not inside the NJC_STATUS_OK check. |
