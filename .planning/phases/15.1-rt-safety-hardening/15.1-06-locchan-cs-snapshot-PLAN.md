<objective>
Replace `m_locchan_cs` audio-thread acquisition (sites at AUDIT lines 1961 and 2698) with an SPSC-mediated snapshot. Audio thread holds a local `LocalChannelMirror[MAX_LOCAL_CHANNELS]`; run/UI thread queues `LocalChannelUpdate` variant records; audio thread drains and applies at the top of AudioProc. Closes AUDIT CR-02.

**Codex HIGH-2 review response (incorporated):** the original plan added a `void* lc_ptr` field to the mirror so the audio thread could call `lc_ptr->m_bq.AddBlock(...)` for the BufferQueue handoff. Codex flagged this as undermining the mirror model — the audio thread still dereferences run-thread-owned objects. **This revision eliminates lc_ptr entirely**: the per-channel `SpscRing<BlockRecord, 16>` (the BufferQueue replacement from 15.1-07b) is stored AS A MEMBER of the LocalChannelMirror itself. The audio thread's `process_samples` loop calls `m_locchan_mirror[ch].block_q.try_push(...)` directly — no pointer dereference into run-thread-owned state.

Note: this plan REQUIRES that the BlockRecord ring lives inside the mirror entry. That is an audio-thread-owned object: it is constructed when LocalChannelAddedUpdate arrives, destructed (drained empty) when LocalChannelRemovedUpdate arrives. The encoder thread (not the audio thread, not the run thread) is the consumer; it drains the ring via a stable handle (the mirror entry's location is fixed inside the NJClient instance for the channel's lifetime). 15.1-07b builds on this by wiring the encoder-thread consumer; this plan only ensures the ring is present and lock-free for the producer side.

**Codex HIGH-3 review response (incorporated):** the original plan identified the lifetime-contract risk in the threat model but did not add a concrete deferred-free task. **This revision adds Task 4** which implements `m_locchan_deferred_delete_q` paralleling 15.1-05's DecodeState pattern: removal is publish-update-first, then enqueue canonical Local_Channel* onto the deferred queue, then drain at the NEXT run-thread tick (gated by `m_audio_drain_generation` atomic that the audio thread bumps each AudioProc call). Run thread waits for generation to advance past the publish before deleting.

    void NJClient::drainLocalChannelUpdates()
    {
        m_locchan_update_q.drain([this](jamwide::LocalChannelUpdate&& upd) {
            std::visit([this](auto&& u) {
                using T = std::decay_t<decltype(u)>;
                if constexpr (std::is_same_v<T, jamwide::LocalChannelAddedUpdate>) {
                    if (u.channel < 0 || u.channel >= MAX_LOCAL_CHANNELS) return;
                    auto& m = m_locchan_mirror[u.channel];
                    m.active  = true;
                    m.srcch   = u.srcch;
                    m.bitrate = u.bitrate;
                    m.bcast   = u.bcast;
                    m.outch   = u.outch;
                    m.flags   = u.flags;
                    m.mute    = u.mute;
                    m.solo    = u.solo;
                    m.volume  = u.volume;
                    m.pan     = u.pan;
                    // block_q is already constructed (in-place inside the array element);
                    // no additional init needed.
                }
                else if constexpr (std::is_same_v<T, jamwide::LocalChannelRemovedUpdate>) {
                    if (u.channel < 0 || u.channel >= MAX_LOCAL_CHANNELS) return;
                    auto& m = m_locchan_mirror[u.channel];
                    m.active = false;
                    // Drain any pending BlockRecords; they will not be processed.
                    // (The encoder thread will see the queue empty and the active flag
                    // false on its next tick; it stops draining this slot.)
                    m.block_q.drain([](jamwide::BlockRecord&&){});
                    // Reset scalar fields. We DO NOT reset the SPSC ring's internal head/tail
                    // by destroying it — the next Add reuses the same ring.
                    m.srcch = 0; m.bitrate = 0; m.bcast = false; m.outch = -1; m.flags = 0;
                    m.mute = false; m.solo = false; m.volume = 1.0f; m.pan = 0.0f;
                }
                else if constexpr (std::is_same_v<T, jamwide::LocalChannelInfoUpdate>) {
                    if (u.channel < 0 || u.channel >= MAX_LOCAL_CHANNELS) return;
                    auto& m = m_locchan_mirror[u.channel];
                    m.srcch   = u.srcch;
                    m.bitrate = u.bitrate;
                    m.bcast   = u.bcast;
                    m.outch   = u.outch;
                    m.flags   = u.flags;

                    if (u.channel < 0 || u.channel >= MAX_LOCAL_CHANNELS) return;

        });
