3. Create `wdl/flacencdec.h` implementing FlacEncoder and FlacDecoder. Follow the research code examples closely:

   **FlacEncoder:**
   - Constructor: Takes (srate, nch, bitrate, serno). bitrate is ignored for FLAC (lossless). serno is ignored.
   - Configure: channels=nch, bits_per_sample=16, sample_rate=srate, compression_level=5, blocksize=1024
   - Init via FLAC__stream_encoder_init_stream() with static write_cb that appends to WDL_Queue outqueue
   - Encode(): Convert float input to FLAC__int32 with clamping [-1.0, 1.0] and scaling by 32767. Handle advance/spacing parameters exactly like VorbisEncoder does (advance is stride between samples, spacing is stride between channels). Call FLAC__stream_encoder_process_interleaved().
   - reinit(): Call FLAC__stream_encoder_finish() + delete + new + reconfigure + init_stream. Clear outqueue first.
   - Available/Get/Advance/Compact: Delegate to outqueue (same as VorbisEncoder)
   - Destructor: finish() then delete()

   **FlacDecoder:**
   - Constructor: Create decoder, init via FLAC__stream_decoder_init_stream() with read_cb, write_cb, metadata_cb, error_cb
   - read_cb: Read from WDL_Queue m_inbuf. If empty, return ABORT (not END_OF_STREAM). Copy available bytes, advance m_inbuf.
   - write_cb: Convert FLAC__int32 samples to float (scale by 1.0f / (1 << (bps-1))), append to WDL_TypedQueue<float> m_outbuf. Interleave channels.
   - metadata_cb: Extract sample_rate and channels from STREAMINFO
   - error_cb: Set error flag, do not crash
   - DecodeGetSrcBuffer(): Return writable area via m_inbuf.Add(nullptr, srclen)
   - DecodeWrote(): Loop calling FLAC__stream_decoder_process_single() while m_inbuf has data and decoder state is OK
   - Reset(): Flush decoder, reset state, clear buffers. Call FLAC__stream_decoder_reset() then re-init.
   - GenerateLappingSamples(): Return 0 (FLAC has no lapping -- NINJAM's overlapFadeState handles crossfade at DecodeState level)

   **Header guard:** Use `#ifndef _FLACENCDEC_H_` / `#define _FLACENCDEC_H_` to match vorbisencdec.h style.

   **Include:** `#include "FLAC/stream_encoder.h"` and `#include "FLAC/stream_decoder.h"`. Also include `"queue.h"` for WDL_Queue.
