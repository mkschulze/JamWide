**This plan is the FIRST of three split from the original 15.1-07** (per Codex HIGH-4 — "07 is too large and too transitional to be safely one sub-plan"):
- **15.1-07a (this plan)**: m_users_cs mirror + RemoteUser deferred-free (CR-01)

- **15.1-07c**: DecodeMediaBuffer SPSC backend (CR-12)

**Codex HIGH-2 review response (incorporated):** the original plan added a `RemoteUser* user_ptr` field to the mirror so the audio thread could access fields not in the mirror. Codex flagged this as undermining the mirror model. **This revision eliminates user_ptr entirely**: every field mixInChannel needs is mirrored BY VALUE. DecodeState* pointers (`chans[ch].ds`, `chans[ch].next_ds[0]/[1]`) are owned by the audio thread once published via PeerNextDsUpdate; ownership transfer is the documented pattern, NOT a back-reference into shared state.

**Codex HIGH-3 review response (incorporated):** the original plan identified the lifetime-contract risk in T-15.1-07-07 but did not add a concrete deferred-free task. **This revision adds Task 4** which implements `m_remoteuser_deferred_delete_q` paralleling 15.1-06 Task 4: peer removal is publish-update-first, then audio-drain-generation gate, then enqueue canonical RemoteUser* onto deferred queue, then drain.

- `SpscRing<RemoteUserUpdate, 64>` per NJClient
- Variant payloads from spsc_payloads.h (15.1-04 final shapes; no user_ptr field per Codex HIGH-2)

    Key visitor branches:
    ```cpp
    void NJClient::drainRemoteUserUpdates()
    {
        m_remoteuser_update_q.drain([this](jamwide::RemoteUserUpdate&& upd) {
            std::visit([this](auto&& u) {
                using T = std::decay_t<decltype(u)>;
                if constexpr (std::is_same_v<T, jamwide::PeerAddedUpdate>) {
                    if (u.slot < 0 || u.slot >= MAX_PEERS) return;
                    auto& m = m_remoteuser_mirror[u.slot];
                    m.active = true;
                    m.user_index = u.user_index;
                    // chans[] elements default to {present=false}; populated by ChannelMaskUpdate later.
                }
                else if constexpr (std::is_same_v<T, jamwide::PeerRemovedUpdate>) {
                    if (u.slot < 0 || u.slot >= MAX_PEERS) return;
                    auto& m = m_remoteuser_mirror[u.slot];
                    m.active = false;
                    // Capture any in-flight DecodeState* pointers and defer-delete them.
                    // (They were owned by the audio thread; cleanup is its responsibility.)
                    for (int ch = 0; ch < MAX_USER_CHANNELS; ++ch) {
                        deferDecodeStateDelete(m_deferred_delete_q, m_deferred_delete_overflows, m.chans[ch].ds);
                        deferDecodeStateDelete(m_deferred_delete_q, m_deferred_delete_overflows, m.chans[ch].next_ds[0]);
                        deferDecodeStateDelete(m_deferred_delete_q, m_deferred_delete_overflows, m.chans[ch].next_ds[1]);
                        m.chans[ch] = RemoteUserChannelMirror{};
                    }
                    // Reset peer-level fields.
                    m.user_index = 0;
                    m.submask = 0; m.chanpresentmask = 0; m.mutedmask = 0; m.solomask = 0;
                    m.muted = false; m.volume = 1.0f; m.pan = 0.0f;
                }
                else if constexpr (std::is_same_v<T, jamwide::PeerChannelMaskUpdate>) {
                    if (u.slot < 0 || u.slot >= MAX_PEERS) return;
                    auto& m = m_remoteuser_mirror[u.slot];
                    m.submask = u.submask;
                    m.chanpresentmask = u.chanpresentmask;
                    m.mutedmask = u.mutedmask;
                    m.solomask = u.solomask;
                    // Update per-channel present/mute/solo flags from the masks.
                    for (int ch = 0; ch < MAX_USER_CHANNELS; ++ch) {
                        m.chans[ch].present = (u.chanpresentmask & (1u << ch)) != 0;
                        m.chans[ch].muted   = (u.mutedmask        & (1u << ch)) != 0;
                        m.chans[ch].solo    = (u.solomask         & (1u << ch)) != 0;
                    }
                }
                else if constexpr (std::is_same_v<T, jamwide::PeerVolPanUpdate>) {
                    if (u.slot < 0 || u.slot >= MAX_PEERS) return;
                    auto& m = m_remoteuser_mirror[u.slot];
                    m.muted = u.muted; m.volume = u.volume; m.pan = u.pan;
                }
                else if constexpr (std::is_same_v<T, jamwide::PeerNextDsUpdate>) {
                    if (u.slot < 0 || u.slot >= MAX_PEERS) return;
                    if (u.channel < 0 || u.channel >= MAX_USER_CHANNELS) { delete u.ds; return; }
                    if (u.slot_idx < 0 || u.slot_idx > 1)                { delete u.ds; return; }
                    auto& chan = m_remoteuser_mirror[u.slot].chans[u.channel];
                    // If a previous next_ds[slot_idx] was set, defer-delete the old one.
                    deferDecodeStateDelete(m_deferred_delete_q, m_deferred_delete_overflows, chan.next_ds[u.slot_idx]);
                    chan.next_ds[u.slot_idx] = u.ds;  // ownership transfer to audio thread
                }
                else if constexpr (std::is_same_v<T, jamwide::PeerCodecSwapUpdate>) {
                    if (u.slot < 0 || u.slot >= MAX_PEERS) return;
                    if (u.channel < 0 || u.channel >= MAX_USER_CHANNELS) return;
                    m_remoteuser_mirror[u.slot].chans[u.channel].codec_fourcc = u.new_fourcc;
                }
            }, upd);
        });
    }
    ```

    Note: `deferDecodeStateDelete` is the helper introduced in 15.1-05. It's reused here.
