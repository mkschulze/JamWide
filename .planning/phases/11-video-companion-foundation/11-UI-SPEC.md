# Phase 11 — UI Design Contract

> Visual and interaction contract for the Video Companion Foundation phase. Two UI surfaces: JUCE plugin additions (video button + privacy modal) and a standalone web companion page (HTML/CSS/JS via Vite). Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (JUCE custom LookAndFeel + vanilla CSS) |
| Preset | not applicable |
| Component library | JUCE LookAndFeel_V4 (plugin), none (companion page) |
| Icon library | Inline SVG (both surfaces) |
| Font | System font stack: `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif` (companion page). JUCE default font (plugin side). |

### Surface Breakdown

This phase has two rendering contexts with matched visual language:

| Surface | Technology | Scope | Primary Focal Point |
|---------|-----------|-------|---------------------|
| JUCE plugin | C++ / JamWideLookAndFeel | Video toggle button in ConnectionBar, privacy/browser-warning modal dialog | Privacy modal dialog (centered, scrim-isolated, demands user decision) |
| Web companion page | HTML / CSS / TypeScript (Vite build) | Branded page with VDO.Ninja iframe grid, session info header, connection status | VDO.Ninja iframe grid (fills 90%+ of viewport height between header and footer) |

## Interaction Contracts

### Video Button Click Flow (JUCE)

```
User clicks "Video" button
  |
  +-- IF not connected to NINJAM server:
  |     Show tooltip "Connect to a server first"
  |     Return (no action)
  |
  +-- IF connected:
        Show Video Privacy Modal (scrim + centered dialog)
          |
          +-- User clicks "Skip Video":
          |     Dismiss modal, return to normal state
          |
          +-- User clicks "I Understand — Open Video":
                1. Dismiss modal
                2. Build companion URL with room/push/noaudio params
                3. Call juce::URL::launchInDefaultBrowser()
                4. Start local WebSocket server (if not already running)
                5. Set Video button text color to kAccentConnect (green/active)
```

### Video Button Re-click (active state)

```
User clicks "Video" while already active (green):
  Re-open companion page in browser (call launchInDefaultBrowser again)
  No modal shown on re-click — privacy notice was shown this session
```

### WebSocket Connection (Companion Page)

```
Page loads
  |
  +-- Show "Connecting to JamWide plugin..." state
  |
  +-- Attempt WebSocket connection to ws://127.0.0.1:{port}
        |
        +-- Success:
        |     Receive "config" message with room, push, noaudio params
        |     Build VDO.Ninja iframe URL
        |     Insert iframe into main area
        |     Update footer: green dot + "Connected to plugin"
        |     Update header: status badge = "Connected"
        |
        +-- Failure / Close:
              Update footer: red dot + "Disconnected"
              Show "Reconnect to Plugin" button in footer
              Update header: status badge = "Disconnected"
              Show centered message: "Connection lost. Click Reconnect to Plugin or re-click Video in JamWide."
```

### Companion Page Responsive Behavior

| Viewport Width | Behavior |
|---------------|----------|
| >= 768px | Header shows logo + title + session info + status badge |
| < 768px | Header hides session info (room ID). Title and badge remain. |
| Any | VDO.Ninja iframe always fills available height. No scrolling. |

## Accessibility

| Concern | Implementation |
|---------|---------------|
| Video button click target | 44px wide x 28px tall (meets 44px minimum touch target width) |
| Modal keyboard focus | Accept button receives initial focus when modal opens |
| Modal escape key | Escape key triggers "Skip Video" (dismiss modal without action) |
| Color contrast — modal body | kTextPrimary (#E0E0E0) on kSurfaceInput (#252840) = 9.3:1 ratio (AAA) |
| Color contrast — button text | kAccentConnect (#40E070) on kSurfaceStrip 20% alpha = 5.2:1 ratio (AA) |
| Companion — status dot | Accompanied by text label (not color-only indicator) |
| Companion — iframe title | `<iframe title="VDO.Ninja video grid">` for screen readers |
