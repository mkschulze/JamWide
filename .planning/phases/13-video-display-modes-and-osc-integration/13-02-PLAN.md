```cpp
void OscServer::handleVideoOsc(const juce::String& address, float value)
{
    // D-14: /JamWide/video/active toggle
    if (address == "/JamWide/video/active")
    {
        if (value >= 0.5f)
        {
            // Activate video -- only if video has been launched this session
            // (privacy modal cannot be shown from OSC context).
            // If already active, this is a no-op (consistent with button re-click behavior).
            // If never launched this session, ignore -- user must click Video button first.
            if (processor.videoCompanion && !processor.videoCompanion->isActive())
            {
                // Check if we have stored session credentials to relaunch
                // If video was previously launched and deactivated, relaunch
                // Otherwise, this is a no-op (user must click Video button first)
                // Use the existing launchCompanion with stored credentials
                auto* nj = processor.njClient.get();
                if (nj && nj->GetStatus() == NJClient::NJC_STATUS_OK)
                {
                    juce::String host = processor.lastServer;
                    juce::String user = processor.lastUsername;
                    juce::String pass = processor.lastPassword;
                    if (host.isNotEmpty())
                        processor.videoCompanion->launchCompanion(host, user, pass);
                }
            }
        }
        else
        {
            // Deactivate video
            if (processor.videoCompanion && processor.videoCompanion->isActive())
                processor.videoCompanion->deactivate();
        }
        return;
    }

    // D-15: /JamWide/video/popout/{idx} trigger
    if (address.startsWith("/JamWide/video/popout/"))
    {
        if (value < 0.5f) return;  // Only trigger on press (1.0)

        juce::String idxStr = address.fromLastOccurrenceOf("/", false, false);
        int oscIdx = idxStr.getIntValue();
        if (oscIdx < 1 || oscIdx > 16) return;  // Bounds check (1-based)

        int userIndex = oscIdx - 1;  // 1-based to 0-based

        if (!processor.videoCompanion || !processor.videoCompanion->isActive())
            return;

        juce::String streamId = processor.videoCompanion->getStreamIdForUserIndex(userIndex);
        if (streamId.isEmpty()) return;  // Index out of roster range

        processor.videoCompanion->requestPopout(streamId);
    }
}
```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| OSC network -> OscServer | UDP OSC messages from any device on the network can control video features |
| OscServer -> VideoCompanion | OscServer calls VideoCompanion methods on the message thread |
| VideoCompanion -> WebSocket clients | WebSocket messages sent to all connected companion page instances |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-13-06 | Spoofing | OSC video/active from unauthorized device | accept | Same threat model as all OSC addresses (Phase 9). OSC is a local-network protocol with no authentication. User configures trusted send/receive IPs. Consistent with IEM plugin pattern. |
