    Replace the send loop (lines ~115-161) with an encryption-aware version. The critical change is: when `m_msgsendpos < 0` and encryption is active, encrypt the payload in-place before building the header:

    In the send loop, after `if (sendm)` and before `if (m_msgsendpos<0)`, add:
    ```cpp

    if (m_msgsendpos < 0 && m_encryption_active && sendm->get_size() > 0) {
        auto enc = encrypt_payload(
            (const unsigned char*)sendm->get_data(),
            sendm->get_size(),
            m_encryption_key
        );
        if (!enc.ok) {
            m_error = -4;  // encryption failure
            break;
        }
        // Replace payload with encrypted version [IV][ciphertext]
        sendm->set_size((int)enc.data.size());
        memcpy(sendm->get_data(), enc.data.data(), enc.data.size());
    }
    ```

## STRIDE Threat Register
