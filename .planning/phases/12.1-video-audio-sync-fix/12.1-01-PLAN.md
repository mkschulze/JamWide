**1. Refactor the delay state model in `companion/src/ui.ts`:**

Replace the single `cachedBufferDelay` variable with a dual-state model:

```typescript
// -- Buffer Delay State (Addresses Codex review HIGH: dual-state model) --
// lastAutoDelayMs: always updated when plugin sends bufferDelay, even in manual mode.
//   This ensures switchToAuto() has a value to restore without waiting for next broadcast.
// activeDelayMs: the value currently applied to the iframe. In auto mode = lastAutoDelayMs.
//   In manual mode = whatever the user set via slider.
// Both start null — no hardcoded 8000ms default. The plugin is the source of truth.
// Pre-config state (both null) means "no delay known yet".
const VDO_NINJA_ORIGIN = 'https://vdo.ninja';

let lastAutoDelayMs: number | null = null;
let activeDelayMs: number | null = null;
let manualMode = false;
```

Remove the old `cachedBufferDelay` variable. Update all references:

- `applyBufferDelay(delayMs)` — Rename to `setLastAutoDelay(delayMs)`. Always sets `lastAutoDelayMs = delayMs`. If NOT in manual mode, also sets `activeDelayMs = delayMs` and calls `sendBufferDelayToIframe(delayMs)`. Logs: `console.log('VideoSync: auto delay updated:', delayMs, 'ms', manualMode ? '(manual mode, not applied)' : '(applied)')`.

- New `switchToManual(delayMs)` — Sets `manualMode = true`, `activeDelayMs = delayMs`, calls `sendBufferDelayToIframe(delayMs)`.

- New `switchToAuto()` — Sets `manualMode = false`, `activeDelayMs = lastAutoDelayMs`, if `lastAutoDelayMs !== null` calls `sendBufferDelayToIframe(lastAutoDelayMs)`.

- New `getLastAutoDelayMs()` — Returns `lastAutoDelayMs`.
- New `getActiveDelayMs()` — Returns `activeDelayMs`.
- New `isManualMode()` — Returns `manualMode`.
- Keep `resetCachedBufferDelay()` but rename to `resetDelayState()` — resets all three to null/false. Update test imports.

- New `getDelayDisplayText()` — Returns formatted string:
  - If `activeDelayMs === null`: return `'--'`
  - If `activeDelayMs === 0`: return `'0.0s (${manualMode ? "manual" : "auto"}) No delay'` (Addresses review MEDIUM: explicit "No delay" label for 0ms)
  - Otherwise: return `'${(activeDelayMs / 1000).toFixed(1)}s (${manualMode ? "manual" : "auto"})'`
