# Plan 14.1-02: Audio Prelisten UI and Editor Wiring

**Objective:** Add Listen/Stop button per server row, volume slider in title bar, active row visual feedback, and wire commands/events through the editor to connect the browser overlay to the prelisten backend.

**Spec:** `docs/superpowers/specs/2026-04-13-audio-prelisten-design.md`

## Interfaces

- `ServerBrowserOverlay::onListenClicked(host, port)` — callback fired on Listen button click
- `ServerBrowserOverlay::onStopListenClicked()` — callback fired on Stop button click
- `ServerBrowserOverlay::setPrelistenState(active, serverName)` — called from editor on event
- `ServerBrowserOverlay::setListenEnabled(enabled)` — disables Listen buttons when in session
- `ServerBrowserOverlay::prelistenVolume` — float synced to processor at 20Hz

- [ ] **1.5** Handle listen button clicks in `listBoxItemClicked()`:

```cpp
void ServerBrowserOverlay::listBoxItemClicked(int row, const juce::MouseEvent& e)
{
    if (row < 0 || row >= static_cast<int>(servers.size()))
        return;

    const auto& entry = servers[static_cast<size_t>(row)];

    int btnX = listBox.getRowPosition(row, true).getWidth() - 70;
    int btnY = listBox.getRowPosition(row, true).getHeight() - 22;
    auto pt = e.getPosition();

    if (entry.users > 0 && (listenEnabled || row == prelistenRow) &&
        pt.x >= btnX && pt.x <= btnX + 62 && pt.y >= btnY && pt.y <= btnY + 16)
    {
        if (row == prelistenRow)
        {
            prelistenRow = -1;
            volumeSlider.setVisible(false);
            volumeLabel.setVisible(false);
            resized(); listBox.repaint();
            if (onStopListenClicked) onStopListenClicked();
        }
        else
        {
            prelistenRow = row;
            volumeSlider.setVisible(true);
            volumeLabel.setVisible(true);
            resized(); listBox.repaint();
            if (onListenClicked) onListenClicked(entry.host, entry.port);
        }
        return;
    }

    if (onServerSelected) onServerSelected(formatAddress(row));
}
```

- [ ] **1.6** Implement `setPrelistenState()` and `setListenEnabled()`:

```cpp
void ServerBrowserOverlay::setPrelistenState(bool active, const juce::String& serverName)
{
    if (!active)
    {
        prelistenRow = -1;
        volumeSlider.setVisible(false);
        volumeLabel.setVisible(false);
    }
    else
    {
        for (int i = 0; i < static_cast<int>(servers.size()); ++i)
            if (juce::String(servers[static_cast<size_t>(i)].host) == serverName)
            { prelistenRow = i; break; }
        volumeSlider.setVisible(true);
        volumeLabel.setVisible(true);
    }
    resized(); listBox.repaint();
}

void ServerBrowserOverlay::setListenEnabled(bool enabled)
{
    listenEnabled = enabled;
    listBox.repaint();
}
```
