<threat_model>
## Threat Model

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Audio thread lock contention from sync state | HIGH | Use std::atomic<bool> for syncWaiting_ and syncActive_ -- never acquire mutex from processBlock. All sync state transitions on audio thread use atomic load/store. |
| Race condition writing m_interval_pos for sync | MEDIUM | SetIntervalPosition() is called ONLY from processBlock, which is the same thread that calls AudioProc. No concurrent access -- naturally serialized. |
| Standalone crash from null AudioPlayHead position data | MEDIUM | Guard all AudioPlayHead field access with has_value() / value_or(). Default hostPlaying=true for standalone (D-16). |
| BPM float comparison false negatives | LOW | Use integer truncation for BPM comparison: static_cast<int>(hostBpm) == static_cast<int>(serverBpm) per JamTaba pattern. |
</threat_model>

**2. Modify `processBlock()` in `JamWideJuceProcessor.cpp`:**

Insert BEFORE the existing AudioProc call (before line 213), after the outputScratch setup:

```cpp
// --- DAW Transport Sync (D-01, D-02, SYNC-01, SYNC-02) ---
bool hostPlaying = true;  // Default: playing (standalone behavior per D-16)

if (auto* playHead = getPlayHead())
{
    if (auto pos = playHead->getPosition())
    {
        hostPlaying = pos->getIsPlaying();

        // Cache host BPM for sync validation on UI thread
        if (auto bpm = pos->getBpm())
            cachedHostBpm_.store(static_cast<float>(*bpm),
                                std::memory_order_relaxed);

        // Sync state machine: WAITING -> ACTIVE transition
        if (syncWaiting_.load(std::memory_order_acquire))
        {
            bool transportJustStarted = hostPlaying && !wasPlaying_;

            if (transportJustStarted)
            {
                // Calculate PPQ sync offset (JamTaba algorithm adapted for JUCE)
                double hostBpm = cachedHostBpm_.load(std::memory_order_relaxed);
                double ppqPos = 0.0;
                double barStart = 0.0;
                int timeSigNum = 4;

                if (auto ppq = pos->getPpqPosition()) ppqPos = *ppq;
                if (auto bar = pos->getPpqPositionOfLastBarStart()) barStart = *bar;
                if (auto sig = pos->getTimeSignature()) timeSigNum = sig->numerator;

                if (hostBpm > 0.0)
                {
                    double samplesPerBeat = (60.0 * storedSampleRate) / hostBpm;
                    int startPosition = 0;

                    if (ppqPos > 0.0)
                    {
                        double cursorInMeasure = ppqPos - barStart;
                        if (cursorInMeasure > 0.00000001)  // Float tolerance (Reaper workaround)
                        {
                            double samplesUntilNextMeasure =
                                (timeSigNum - cursorInMeasure) * samplesPerBeat;
                            startPosition = -static_cast<int>(samplesUntilNextMeasure);
                        }
                    }
                    else
                    {
                        startPosition = static_cast<int>(ppqPos * samplesPerBeat);
                    }

                    // Apply offset to NJClient interval position
                    int iPos, iLen;
                    client->GetPosition(&iPos, &iLen);
                    if (iLen > 0)
                    {
                        int newPos;
                        if (startPosition >= 0)
                            newPos = startPosition % iLen;
                        else
                            newPos = iLen - std::abs(startPosition % iLen);
                        client->SetIntervalPosition(newPos);
                    }
                }

                syncWaiting_.store(false, std::memory_order_release);
                syncActive_.store(true, std::memory_order_release);
            }

            // While WAITING, send silence (D-04)
            hostPlaying = false;
        }
    }
}

// Edge detection (audio-thread-only, no sync primitive needed)
wasPlaying_ = hostPlaying;
```
